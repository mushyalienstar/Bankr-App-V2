# Bankr-App
Simple demo bank transaction app where you can view your balance, transfer money between accounts, and close your account.

As this was created specifically to learn JavaScript, it is generally quite simple (static) and requires the user to sign up and log in with a name and a four-digit password.

After the user has signed in and made a username and password, they have to log in with the same username and password they signed in with.

The transactions and balance are also preset at the start as a template, but can be added to using the app's features.

You can also interact with two other premade accounts (Username: "John Doe" - Canadian Account) & (Username: "Sarah Doe" - Japanese Account) by transferring money from your own account or by logging in as that user by pressing the buttons in the top-left corner.
